# Marimba-Web

## Documentation
To view documentation, install docfx command-line following instructions on 
[https://dotnet.github.io/docfx/tutorial/docfx_getting_started.html](https://dotnet.github.io/docfx/tutorial/docfx_getting_started.html).
Then, navigate to the **Documentation** directory from a terminal, and run `docfx --serve` to start the a webpage running on localhost:8080.
Then, navigate to `localhost:8080/api/index.html` to view the documentation.

Alternatively, if you want to generate a pdf, run `docfx` from the **Documentation** directory and a pdf with the documentation will be generated
